## 三、核心文件说明

### 3.1 `docs/index.html` — 主页

页面结构（从上到下）：

| 区块 | 说明 |
|------|------|
| **导航栏** | 主页 / API 聚合导航 / AI 探索 / 文章分享 |
| **Hero** | 姓名、标题、简介 + 三栏信息 + 配图 |
| **AI 探索** | 分类筛选标签 + 资源卡片网格 + "显示更多"按钮 |
| **文章分享** | 文章列表 + "显示更多"按钮 |
| **Footer** | 姓名 + 回到顶部 |

### 3.2 `docs/js/main.js` — 数据配置中心

这是**最重要的文件**，几乎所有的内容修改都在这里：

```javascript
const CONTENT = {
    // 导航、Hero 文字
    // aiResources: [...]   ← AI 探索资源卡片数据
    // articles: [...]      ← 文章分享数据
}
```

- **修改文字**：直接改 `CONTENT` 对象里的字段
- **新增 AI 资源**：在 `aiResources` 数组里添加对象，字段：`title`、`description`、`tags`、`link`、`category`
- **新增文章**：在 `articles` 数组里添加对象，字段：`date`、`title`、`summary`、`content`（内联模式）、`link`（外链模式）、`external`

## 四、如何添加内容

### 4.1 添加 AI 资源卡片

编辑 `docs/js/main.js`，在 `aiResources` 数组中插入：

```javascript
{
    title: '资源名称',
    description: '简短描述',
    tags: ['标签1', '标签2'],
    link: 'https://...',        // 外部链接
    // 或 link: 'articles/xxx.html'   // 站内页面
    category: '大模型'           // 分类：大模型 / AI工具 / 提示词 / 开源
}
```

### 4.2 添加文章

编辑 `docs/js/main.js`，在 `articles` 数组中插入：

**外链模式**（跳转到站内 HTML 页面或外部链接）：
```javascript
{
    date: '2026.06',
    title: '文章标题',
    summary: '文章摘要',
    content: [],
    link: 'articles/xxx.html',   // 或外部 URL
    external: true
}
```

**内联模式**（点击展开阅读全文）：
```javascript
{
    date: '2026.06',
    title: '文章标题',
    summary: '文章摘要',
    content: ['段落1', '段落2', '段落3'],
    link: '',
    external: false
}
```

### 4.3 创建独立的文章/资源页面

参照 `docs/articles/thought-partner-prompt.html` 的模板结构：
- 引入 `../css/custom.css`
- 使用标准 header + `.article-page` + `.site-footer`
- 样式写在 `<style>` 标签中

## 六、常用操作

| 操作 | 方法 |
|------|------|
| 修改 Hero 文字 | 编辑 `main.js` 中 `CONTENT` 的对应字段 |
| 调整默认显示数量 | 修改 `main.js` 中 `INITIAL_SHOW_COUNT`（资源）和 `INITIAL_ARTICLE_SHOW`（文章） |
| 修改样式 | 编辑 `custom.css` 中的 CSS 变量即可全局生效 |
| 添加新页面 | 在 `docs/articles/` 下新建 HTML，参考已有页面的模板结构 |
| 预览 | 双击 `docs/index.html` 在浏览器打开 |
