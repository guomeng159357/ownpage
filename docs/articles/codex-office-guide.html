将 Codex 融入日常办公：从入门到精通

从零开始，逐步把 Codex 变成你的得力办公助手。涵盖晨间简报、文件整理、PPT 生成、数据分析等实战场景，附完整学习路径。

将 Codex 融入日常办公，最有效的方式是从一个你每天都会做的、具体且重复的工作场景开始，比如处理邮件、整理文件或撰写周报。不要一开始就想解决大问题，而是专注于让一个小任务变得更轻松。

下面这条从易到难的学习路径，能帮你轻松上手，逐步把 Codex 变成你的得力办公助手。

🚀 阶段一：从零开始，完成你的第一个自动化任务

预计时间：1 小时内。目标是跑通流程，建立信心。

场景一：让"晨间简报"成为你的 AI 秘书（入门首选）

许多信息工作者每天早上的第一件事，就是"搞清楚今天要干什么"。这正是学习使用 Codex 最完美的切入点。

Level 1 — 信息汇总

创建一条新对话，直接输入指令。Codex 会自动连接你的日历、邮件和通讯软件。

连接我的 Calendar、Gmail 和 Slack，告诉我今天有哪些会议、待回复的邮件和未读的重要信息。

Level 2 — 固定格式

当信息汇总变得有用后，你可以把偏好固定下来，避免每次重复说明。告诉 Codex：

把今天的汇总格式保存到我的 AGENTS.md 文件里，以后每次都用这个格式。

场景二：整理杂乱的"下载文件夹"

这是最直观、最能立即见效的办公任务。

在 Codex 桌面应用中，将你杂乱的"下载"文件夹添加为一个"项目"。然后在对话中输入：

帮我把当前目录下的文件按类型整理到不同文件夹里：图片放 images，文档放 docs，压缩包放 archives，其他放 others。
💡 学习要点：通过这个任务，你会理解 Codex 的"项目"概念——它不是一个聊天窗口，而是能真正操作你电脑上文件和文件夹的代理。

📈 阶段二：掌握核心技能，让协作更高效

预计时间：半天到一天。在体验过基础任务后，学习以下核心技能能让你事半功倍。

学会"先计划，后执行"

对于稍复杂的任务（如制作 PPT 或分析表格），先让 Codex 给出方案，可以极大避免无效返工。

我想做一份关于 Q2 销售数据的分析报告，先不要动代码。请给我一个数据清洗、分析和可视化的三步计划，我确认后再开始。

建立你的"项目宪法" AGENTS.md

这是与 Codex 沟通的关键。在项目文件夹根目录创建一个 AGENTS.md 文件，把你反复交代的规则写进去，比如"使用中文回复"、"代码注释用英文"、"图表风格用'蓝白商务风'"。这样，Codex 就会自动遵守你的个人习惯。

巧用"计划模式"和"预览"

当你让 Codex 做网页、PPT 或报告时，在对话中输入 /plan 或手动开启"计划模式"。它会先给出详细实施方案等你确认，避免跑偏。对于网页、图表等可视化结果，可以使用内置的预览窗口。如果对布局不满意，直接用批注功能在页面上圈出来并写下意见，比用文字描述精准得多。

💼 阶段三：解锁办公进阶场景，成为效率高手

当你掌握了基础操作，就可以用 Codex 来应对更复杂的办公挑战了。

场景一：AI 一键生成专业 PPT

这是最受欢迎的办公功能之一。利用 $$slides 技能，Codex 能直接生成可编辑的 .pptx 文件。

使用 $$slides 技能，帮我创建一个蓝色风格、白色底的PPT，主题是"2026下半年市场策略"，大纲包括：市场回顾、核心挑战、策略规划、预算预估。总页数控制在8页。

场景二：自动化处理数据与制作报告

无论是分析销售数据还是做市场调研，Codex 都能完成从数据清洗到生成可视化报告的全流程。

请分析我提供的"销售数据.csv"文件。1. 清洗数据，去除无效条目。2. 分析各产品线的季度销售趋势。3. 生成一份包含柱状图和趋势线的 HTML 可视化报告。

场景三：用 @Computer 控制电脑完成任务（Beta）

这是更进阶的玩法，让 Codex 像人一样操作你的电脑界面，适合跨应用的复杂任务。

@Computer 请帮我把桌面上的「会议笔记.txt」里的待办事项，添加到我的飞书"今日待办"清单里。
💡 使用提示：首次使用需要在 Codex 应用中开启 Computer Use 功能，并在指令前加上 @Computer。

💡 进阶心法：让它成为你的"虚拟员工"

当你熟练之后，可以像 OpenAI 的专家一样，将 Codex 提升到"虚拟员工"的层次。

给一个长期目标 /goal

对于周期较长的任务，比如"分析这个月的客户反馈邮件，总结出 3 个主要痛点，并生成一份改进建议报告"，使用 /goal 模式，Codex 会像一个项目经理一样，自己规划步骤并持续工作直到完成。

让它拥有"长期记忆"

用本地文件夹（如 Obsidian 知识库）来存放项目文档、会议记录和 TODO 清单，并在 AGENTS.md 中告诉 Codex 每次执行任务前先去读取这些文件。这样，它能记住项目的来龙去脉，变得越来越"懂你"。

开启"自动化"

打开 Codex 的"自动化"面板，把你觉得有用且重复的任务设置成定时执行。例如，设置一个"每个工作日上午 9 点，自动整理昨天的项目文件并生成进度摘要"。

📚 持续学习与资源

  • 从模仿开始：不用担心写不出完美的指令。先去 OpenAI 官网看看他们提供的真实世界用例，这些是最好的学习模板。
  • 遵循小步快跑：始终遵循"只调研 → 单文件改动 → 小任务修复 → 修复+验证 → 多文件改动"的阶梯，逐步建立你对工具的信任和掌控力。
纸上得来终觉浅，自己动手跑一遍才是学习 Codex 最快的方式。你可以先从"整理桌面"或"生成一份晨间简报"开始，相信你会很快感受到它的魅力。
